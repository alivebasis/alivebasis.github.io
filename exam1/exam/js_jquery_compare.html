Javascript vs Jquery

Javascript

  1. 크로스브라우징에 약하다!
  2. 제이쿼리에 비해 코드 구문이 복잡하다.
  3. [Selector] children vs childNodes
    • children은 요소 자식만 포함된다. (IE9+)
    • childNodes는 주석, 텍스트 등까지 모두 포함된다.
  4. [Selector] 모든 노드에 주석, 텍스트가 포함된다.

JQuery

  1. 크로스브라우징에 강하다!
  2. 자바스크립트에 비해 속도는 느리나, DOM 접근성과 성능면에서 좋다.
  3. 요소 선택(Selector)이 자바스크립트보다 간단하고, 다양하다.
